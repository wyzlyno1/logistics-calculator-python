# 物流计算工具箱 — Python 版

面向物流工程专业的综合性 Web 计算平台，覆盖库存管理、物流规划、需求预测等核心算法。基于 Streamlit + Plotly，纯 Python 实现。

## 功能模块

### 库存管理
| 模块 | 说明 |
|------|------|
| 经济订货批量 (EOQ) | 基本 EOQ、数量折扣分析、允许缺货模型 |
| 经济生产批量 (EPQ) | 生产批量优化、最大库存计算 |
| 安全库存计算 | 需求不确定 / 提前期不确定 / 两者均不确定 |
| ABC 分类分析 | 帕累托图、自定义分类阈值 |

### 物流规划
| 模块 | 说明 |
|------|------|
| 重心法选址 | 基本重心法 + 迭代优化 |
| 选址案例分析 | 因素评分法 + 盈亏平衡法 |
| 运输问题求解 | 最小元素法、伏格尔法、位势法、闭回路法 |
| 物流网络优化 | Dijkstra、Prim、Ford-Fulkerson |

### 需求管理
| 模块 | 说明 |
|------|------|
| 需求预测方法 | 移动平均、加权移动平均、指数平滑 + 误差分析 |
| 牛鞭效应演示 | 四级供应链动态仿真 |

## 技术栈

- **框架**: Streamlit
- **图表**: Plotly
- **图算法**: NetworkX
- **数据处理**: Pandas + NumPy

## 使用方式

```bash
pip install -r requirements.txt
streamlit run app.py
```
